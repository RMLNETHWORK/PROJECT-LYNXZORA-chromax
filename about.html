About ChromaX

pick. convert. copy.

ChromaX is a free, browser-based color converter and harmony generator. Pick a color once and instantly see it expressed in every format a designer or developer actually needs — no sign-up required to use.

Supported formats

  • HEX — the web-standard hexadecimal color code
  • RGB — red, green, blue channel values
  • HSL — hue, saturation, lightness
  • HSV — hue, saturation, value/brightness
  • LAB — perceptually uniform CIE Lab color space
  • CMYK — cyan, magenta, yellow, key (for print workflows)
  • OKLCH — modern perceptual color space gaining CSS support
  • CSS variable — ready-to-paste --color: declarations

Color harmonies

Beyond single-color conversion, ChromaX generates complementary, analogous, triadic, and other classic color harmonies from whatever color you're working with, so you can build a palette instead of guessing at one.

Live previews

Because a color rarely lives in isolation, ChromaX includes live e-commerce and blog UI mockups that re-theme in real time as you adjust your color, giving you a rough sense of how it will actually read in a real interface — light or dark.

Accessibility

ChromaX includes a colorblindness (CVD) simulator so you can check how your chosen color appears under common types of color vision deficiency before you ship it.

Who made this

ChromaX is a portfolio project by Rolance Labayog @ LynxZora. It's built with plain HTML, CSS, and JavaScript — no frameworks, no build step — as a hand-rolled exercise in color math and interface design.
